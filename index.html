    meta 元素
    HTML meta 元素表示那些不能由其他 HTML 元相关（meta-related）元素表示的元数据信息。 如： base link script style 或 title
    meta 元素定义的元数据的类型包括以下几种：
      name 指定了 meta 元素的类型；说明该元素包含了什么类型的信息。
      content 指定了实际的元数据内容。
        常见meta-name值
        1. charset="UTF-8" 规定了字符编码
        2. name="author" content="ygenging" 规定了页面作者名字
        3. name="description" content="页面描述" 有利于搜索引擎搜索
          （这些行为在术语上被称为：搜索引擎优化 或 SEO）。
    link rel="icon" href="images/icon.png" 页面头像
